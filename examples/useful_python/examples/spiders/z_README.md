# 心得
* bytes类型中存的就是每个字符对应的编码，只不过print的时候自动显示成了对应的ASCII字符



# 常用代码片段
* bytesarray 转 string
    ```python
    b = bytes('我', encoding='utf8')
    c = [*b]
    # 这里的c实际就为对应的ASCII编码对应的数组
    bytearray(c).decode(encoding="utf8")
    ```
